SIMON WILLISON + 精选 61

用 LLM 写 SQL 的实战经验：为什么 schema 注释比 few-shot 更管用

Simon Willison 在博客中分享近三个月用 LLM 生成 SQL 查询的实战观察。核心发现：在 schema 字段上添加详细 COMMENT（包括字段含义、枚举值范围、与其他表的隐式关系）比在 prompt 中提供 5-10 个 few-shot 示例的准确率更高约 15%；模型在没有 schema 注释时倾向于"幻觉"出表里不存在的字段名；推荐流程：先让 LLM 阅读 DDL + COMMENT、明确"未知字段时主动询问而非猜测"，再让其生成查询。文中给出 Postgres、SQLite、DuckDB 三种数据库的具体 COMMENT 实践示例。
